Price Bee - Product Comparison Platform

The Product Comparison Platform is designed to facilitate the evaluation of various products through a user-friendly web application. Built using Angular for the front-end and a .NET API for the back-end, this system enables users to compare product features, prices, and specifications seamlessly.

Key Features:
  • Dynamic User Interface: Angular provides a responsive and interactive user experience
  • Robust API Integration: .NET API manages data retrieval and processing
  • Product Management: Add, update, and remove products from comparison lists
  • Real-Time Updates: Instantaneous updates on product information
  • Search and Filter Options: Advanced search and filtering capabilities

This platform empowers consumers to make informed purchasing decisions by providing comprehensive product comparisons in a streamlined manner.

  • Angular
  • ASP.NET Web API
  • Entity Framework
  • Microsoft SQL Server

Attendance Tracking System

The Comprehensive Attendance/Time Tracking System is a robust web application developed using ASP.NET Core MVC, SQL Server, EF Core Code First, HTML, JavaScript, jQuery, and Bootstrap. This system streamlines attendance monitoring for students, instructors, and employees.

Key Features:
  • User Management: Complete user and profile management system
  • CRUD Operations: Full CRUD operations for students and departments
  • Permission System: Advanced permission request handling
  • Daily Attendance: Automated daily attendance recording
  • Schedule Management: Comprehensive schedule management tools
  • Analytics: Valuable insights into attendance trends

This system serves as an essential tool for educational institutions and businesses, providing comprehensive attendance monitoring and analytics.

  • ASP.NET Core MVC
  • SQL Server
  • Entity Framework Core
  • JavaScript & jQuery
  • Bootstrap
